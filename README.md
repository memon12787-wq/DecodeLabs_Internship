## Project 2 — Iris Flower Classification
A machine learning classification model built using the Iris dataset. The project uses the K-Nearest Neighbors (KNN) algorithm to classify iris flowers into different species based on their measurements.

### Concepts
- Data Loading and Preprocessing
- Train-Test Split
- Supervised Learning
- K-Nearest Neighbors (KNN)
- Model Evaluation
- Classification Prediction
### Features
- Loads and analyzes the Iris dataset
- Splits data into training and testing sets
- Trains a KNN classification model
- Evaluates performance using accuracy score
- Generates a classification report and confusion matrix
- Predicts the species of a new flower sample
### Run
Iris_Classification_Model.py
